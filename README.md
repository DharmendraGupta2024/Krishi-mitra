# Krishi Mitra

## 🌾 Your Complete Digital Companion for Modern Agronomy

Krishi Mitra is a comprehensive web platform designed to empower farmers and agriculture enthusiasts with essential information, modern farming techniques, government schemes, and technological insights. The platform serves as a one-stop solution for all agricultural needs, from weather forecasting to crop management and sustainable farming practices.

## ✨ Features

### 🌤️ Weather Information
- Real-time weather updates and forecasts
- Location-based weather tracking
- Current location weather service

### 🏛️ Government Schemes
- Comprehensive database of agricultural government schemes
- Easy access to subsidy information and application processes
- Policy updates and farmer welfare programs

### 🌱 Crop Management
- Detailed information on various seeds and crops
- Real-time seed price tracking and updates
- Fertilizer recommendations and usage guidelines
- Crop-specific farming techniques

### 🔧 Technology Integration
- Solar panel information for sustainable farming
- Modern agricultural technology insights
- Smart farming techniques and tools

### 📝 Feedback System
- User feedback collection and analysis
- Continuous improvement based on farmer input

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Frameworks & Libraries**:
  - Bootstrap for responsive design
  - jQuery for DOM manipulation
  - Owl Carousel for image sliders
  - Revolution Slider for hero sections
  - Font Awesome for icons
- **Styling**: Custom CSS with modern UI components
- **Fonts**: Google Fonts (Outfit family)

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for weather data and external resources

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/krishi-mitra.git
   cd krishi-mitra
   ```

2. **Open the application**
   - Open `index.html` in your web browser
   - Navigate through different sections using the menu

### Usage

1. **Homepage**: Start with the main dashboard
2. **Weather**: Check weather conditions and forecasts
3. **Government Schemes**: Browse available schemes and subsidies
4. **Crop Information**: Access seed and fertilizer details
5. **Technology**: Learn about modern farming technologies
6. **Feedback**: Share your experience and suggestions

## 🔧 Development Scripts

The project includes Python scripts for data management:

- `update_seedprice.py`: Updates seed pricing information
- `update_solar.py`: Updates solar panel data
- `fix_styles.py`: CSS styling utilities
- `upgrade_detail.py`: Content management
